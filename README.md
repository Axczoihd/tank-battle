# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas 和 JavaScript 制作的双人坦克对战游戏。

## 🎯 游戏玩法

- **双人对战**：两位玩家在同一屏幕上操控各自的坦克进行对战
- **目标**：先获得 5 分的玩家获胜
- **障碍物**：战场上分布着墙壁，可以用来躲避和制定战术
- **无敌时间**：重生后有短暂的无敌保护时间

## 🕹️ 操作方式

| 操作 | 玩家一 (🟢) | 玩家二 (🔴) |
|------|------------|------------|
| 移动 | W A S D | ↑ ↓ ← → |
| 射击 | 空格键 | Enter 键 |

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。

```bash
# 克隆仓库
git clone https://github.com/Axczoihd/tank-battle.git

# 打开游戏
cd tank-battle
open index.html  # macOS
# 或在 Windows 上直接双击 index.html
```
